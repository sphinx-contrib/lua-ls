Sphinx-LuaLs
============

Sphinx-LuaLs features domain for `Lua`_ and automatic documentation generator
based on `EmmyLua`_ or `LuaLs`_ language servers.

.. _Lua: https://lua.org
.. _EmmyLua: https://github.com/EmmyLuaLs/emmylua-analyzer-rust/
.. _LuaLs: https://lua_ls.github.io/

See an example output: `logging`.
